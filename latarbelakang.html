Keahlian Saya

Desain Web HTML

Saya memiliki keahlian dalam membuat desain web yang responsif dan menarik menggunakan HTML. Saya memastikan bahwa setiap elemen web yang saya buat memiliki struktur yang baik dan mudah dinavigasi oleh pengguna.

JavaScript

Saya ahli dalam JavaScript dan mampu membuat fitur-fitur interaktif pada situs web. Dengan JavaScript, saya dapat menambah fungsi dinamis yang meningkatkan pengalaman pengguna, seperti validasi formulir, animasi, dan pengelolaan data.

CSS

Saya memiliki keahlian dalam CSS yang memungkinkan saya untuk membuat desain web yang estetis dan konsisten di berbagai perangkat. Saya menguasai teknik seperti Flexbox dan Grid untuk tata letak yang fleksibel dan efisien.
